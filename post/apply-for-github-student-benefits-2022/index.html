Github Student Benefits申请相关（2022年版

2022-02-09 GitHub 冲浪分享

看到有fedi友在说这个嘟文链接

前两周刚申请的我说一下需要的材料（因为网上的教程基本上已经不再实用了）（仅针对在校大学生）
首先是需要去GitHub里绑定你学校的邮箱（不知道学校邮箱的可以去你们学校的官网上找找，有学校邮箱还可以薅apple music四年的羊毛），绑定完成后点开申请页面github education 进行登录，登录之后点benefits会有一个申请界面，选择你的教育邮箱（一般情况下会自动识别出你的学校，如果没有再自己手动填写），下面会有一个要你说你打算用GitHub干啥的问答。我写了一下自己的情况然后大概说了想用GitHub搭博客和别人交流等等，怕自己英文水平不行的直接上翻译器没太大毛病就行。

第二步是根据你的IP地址判定你在不在学校，不在学校会让你提交一个学生证之类的证明你的学生身份。界面上只有用电脑摄像头拍照但是你可以用手机拍完后存在电脑上再把照片拖进去也是OK的。底下有个项目，你传了啥选啥（基本上都是第一个）。
需要证明你的学校、入学时间。
因为要上传个人信息所以请仔细斟酌是否需要申请。

最后说一下如果申请没过怎么办。申请过后接着戳benefits界面，右侧是你的申请状态，如果按照规定上传了但是被reject了请直接点底下的support-community-education-contact us提交说明。我大概说的就是我按照你们的要求提交了教育邮箱和其他材料进行申请但是你们把我拒绝了，我觉得自己得到了不公平的对待，请问能否核查我的学生资格（对不起还是直接丢的翻译器）。第二天就给我发邮件道歉并且通过了学生认证。

以上仅供参考，总之就是如果申请不过请大胆地去找support进行提问，不要害怕！你是年满13周岁的学生！这是你可以享受的！

另外附上huhuhang/education-discount: 适合中国大陆学生的优质教育优惠项目 和ivmm/Student-resources: 本文介绍的是利用学生、教职工身份可以享受到的相关学生优惠、教育优惠或教师优惠的权益，但也希望各位享受权利的同时不要忘记自己的义务，不要售卖、转手自己的学生优惠、教育优惠的资格，使得其他同学无法受益。，朋友们，能薅则薅
